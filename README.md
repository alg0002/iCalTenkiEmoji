iCalTenkiEmoji
==============

## これは何？

livedoor天気情報の[iCal天気][1]を参照して、iOS端末の標準カレンダーに絵文字っぽく天気情報を表示する


## 何ができる？

[iCal天気][1]をそのまま参照すると   
![iCal天気を参照した場合](http://icaltenkiemoji.azurewebsites.net/img00.png)
こうなります。

iCalTenkiEmojiを参照すると   
![iCalTenkiEmojiを参照した場合](http://icaltenkiemoji.azurewebsites.net/img01.png)
こうなります。
